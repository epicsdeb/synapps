ccd Release Notes

NOTE: This module is now obsolete and has been replaced by the areaDetector module.

Release 1-10 (2-Feb-2009)

  • Added code to support parallel make.
  • Use new "busy" module for busy record support, not in sscan module any longer.
  • NOTE: The new version of roperServer in R1-9 and R1-10 may not work on some Windows systems. The older server from R1-8 and before can still be used in this case.

Release 1-9 (5-Mar-2008)

  • Converted roperServer from Visual Basic 6 to Visual Basic .NET 2003.
  • Fixed bug in roperServer that caused it to treat 16-bit data as signed rather than unsigned.

Release 1-8 (9-Oct-2007)

  • Fixed logic in roperCCD.st to eliminate race condition on AcquireCLBK when saving files with SaveFile rather than with AutoSave. Before this fix one needed a delay (~0.5 second) after SaveFile before starting the next frame or acquisition would not start.
  • Changed the SaveFile record type from bo to busy so that it can be used with ca_put_callback if desired and EPICS will wait until the file save is complete.

Release 1-7 (4-Oct-2007)

  • Added new FrameType, "Dbl correlation" to collect frames with double correlation. This collects 2 frames, each with half the requested exposure time, and removes cosmic rays (zingers) by comparing the frames. This option is currently only implemented on the marCCD.
  • Added start_epics and sync_adls in iocMarCCD

Release 1-6 (6-Sep-2005)

  • Added support for WIN32.
  • Removed variable argument macros for debugging in SNL code.
  • Minor changes in MAR code.

Release 1-5 (31-Mar-2005)

  • Added rpc library dependency for cygwin.
  • Changed from using SaveAs to Save in roperServer, so that default file type (SPE, TIFF, etc.) is preserved.

Release 1-4 (23-Mar-2005)

  • Changes for asyn, particularly with handling terminators
  • Fixed timing problem with EPICS shutter with roperCCD.st

Release 1-3 (30-Jun-2004)

  • Removed MPF, old socket library, and IP, all replaced with ASYN
  • Support double correlation for backgrounds in marCCD

Release 1-2 (18-Feb-2004?)

  • Added SSCAN for busy record
  • Added autosave for save/restore

Release 1-1 (27-Jan-2004)

This is the first release of the synApps ccd module.  The smartControl program was previously available on the CARS Web site.  The other software is new with this release.  This version runs only on EPICS 3.14.

The support for the MAR-165 CCD and the Roper CCD detectors is currently complete and functional.

The support for the Bruker CCD detectors (via the smartService socket interface) is not complete or functional.

The support for the MAR-345 online image plate is not complete or functional.
