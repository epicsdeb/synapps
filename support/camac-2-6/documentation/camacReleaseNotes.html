  • Minor changes for conversion from CVS to SVN, and parallel make support.

  • Removed Debug macros with variable number of arguments

Release 2-4, March 29, 2005

  • Changed drvMotorE500.cc to use generic epicsThread priority

Minor changes:

  • Removed dependency on MPF in configure/RELEASE
  • Fixed dset definition in devScalerCamac.c
  • Change in parameter passing and stack size definition in drvMotorE500.cc
  • Updated configure/CONFIG_APP and iocBoot/Makefile for 3.14.7

Release 2-2

Minor changes:

  • Added calc and sscan to configure/RELEASE
  • Changed CALLBACK definition in devScalerCamac.c
  • Changes to drvMotorE500.cc for limits and status passing

Release 2-1

Conversion to EPICS 3.14.

  • Removed dxp software for XIA DXP modules.  Moved it to a new application called dxp.
  • Added documentation directory

Release 1-3

Fixed bug in dxpRecord.c, missing one task parameter.

Release 1-2

Added debugging to ksc2917.h.

Updated limit switch logic in drvMotorE500.c

Changed connectIO to connectWait in devDXP.cc.  Requires mpf1-10.

Added GAIN, PGAIN, EMAX, FIPPI, EMAXRBV, GAINRBV fields to dxp record.

Added support for downloading new FiPPI code at run time.

Added iocxxx directory.
